# pgAdmin 4: Professional DB Management

**pgAdmin 4** is the most popular and feature-rich open source administration and development platform for PostgreSQL.

## Why use this?
- **Full Control**: Create tables, users, and schemas with a powerful GUI.
- **Query Tool**: Write and execute complex SQL/Spatial queries with syntax highlighting and explain plans.
- **Geometry Viewer**: Use the built-in geometry viewer to visually inspect PostGIS query results on a map without leaving the interface.

## Quick Start
- **Connection**: Use the "Add Server" dialog to connect to `gis-postgres` using your configured credentials.
- **Web Interface**: Accessible securely on port **5050**.
