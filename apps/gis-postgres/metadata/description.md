# GIS PostGIS Engine

This app provides a robust **PostgreSQL 16** database pre-loaded with **PostGIS 3.4** and **pgvector**. 

It serves as the central engine for your GIS catalog, powering mapping (Lizmap/Martin) and AI (AnythingLLM) applications.

## Features
- PostgreSQL 16
- PostGIS 3.4 (Geospatial)
- pgvector (AI/Vector Embeddings)
- Standard connection on port 5432

## Usage
Connect to this database using the credentials you configured. Other Runtipi apps can connect using the hostname `gis-postgres`.
