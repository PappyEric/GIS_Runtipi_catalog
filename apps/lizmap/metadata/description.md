# Lizmap Web Client

Lizmap allows you to publish QGIS projects as interactive web maps directly from QGIS Desktop to the Web.

## Stack
This app bundles:
- **Lizmap Web Client**: The frontend interface.
- **QGIS Server**: The backend rendering engine.
- **Redis**: For session and cache management.

## Usage
1. Upload your QGIS projects (`.qgs` or `.qgz`) and their config files (`.qgis.cfg`) to the data directory.
2. Access the Lizmap interface to view your maps.
